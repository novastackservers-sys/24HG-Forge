Alternative install methods

Already on Fedora Atomic? Rebase directly — no ISO needed.

Rebase from Bazzite or any Fedora Atomic

Run this single command to replace your current OS image with HubOS:

rpm-ostree rebase ostree-image-signed:docker://git.raggi.is/24hg/hubos:latest

Then reboot:

systemctl reboot
Note: Rebasing preserves your home directory and user data but replaces the OS layer entirely. Your installed Flatpak apps are also kept.
