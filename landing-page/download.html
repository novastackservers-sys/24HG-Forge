Alternative install methods

Already on Fedora Atomic? Rebase directly — no ISO needed.

Rebase from Bazzite or any Fedora Atomic

Run this single command to replace your current OS image with 24HG:

# AMD/Intel GPU: rpm-ostree rebase ostree-unverified-registry:ghcr.io/novastackservers-sys/24hg:latest-amd # NVIDIA GPU: rpm-ostree rebase ostree-unverified-registry:ghcr.io/novastackservers-sys/24hg:latest-nvidia

Then reboot:

systemctl reboot
Note: Rebasing preserves your home directory and user data but replaces the OS layer entirely. Your installed Flatpak apps are also kept.
