Already on Fedora Atomic? Rebase Instead

Running Bazzite, Silverblue, or Kinoite? Switch to HubOS in one command — no reinstall needed.

Desktop (AMD / Intel GPU)

rpm-ostree rebase ostree-unverified-registry:registry.24hgaming.com/hubos-desktop:latest

NVIDIA

rpm-ostree rebase ostree-unverified-registry:registry.24hgaming.com/hubos-nvidia:latest

Steam Deck / Handheld

rpm-ostree rebase ostree-unverified-registry:registry.24hgaming.com/hubos-deck:latest

Then reboot:

systemctl reboot
What is rebasing? It replaces your OS image with HubOS while keeping your user data. It's atomic — if anything fails, your current system is untouched. Roll back anytime with rpm-ostree rollback.

After Install

Verify you're running HubOS:

cat /etc/os-release | grep PRETTY_NAME

Should output: PRETTY_NAME="HubOS (24 Hour Gaming)"

Run diagnostics:

hubos-diag
